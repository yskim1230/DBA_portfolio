# Start Here

## 1. 요약
저는 8년차 DBA로 온프레미스 MSSQL, MySQL, MariaDB 운영과 클라우드 NCP Cloud DB for MySQL 운영 전환 업무를 수행했습니다.
운영을 감으로 하지 않고 절차와 증적 중심으로 표준화하는 방식으로 일합니다.
이 저장소는 제가 만든 런북, 스크립트, 로그, 정량 지표를 기반으로 재현 가능한 형태로 정리한 포트폴리오입니다.
